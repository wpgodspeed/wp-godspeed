# Using WP Godspeed Securely

This document discusses how to safely deal with a potentially compromised API key. Below, we also provide guidelines on how to change your API key.

## The WP Godspeed API key

### How is the API key used?

For every site, an API key is necessary in order to securly communicate with the WP Godspeed API. An API key is issued during plugin registration. The key is stored in the local WordPress database. For all communications, the key is retrieved from the database, base64 encoded, and then sent to the server as an `Authorization` header via SSL protocols.

### Make API key change requests

If your WordPress site has for any reason been compromised, it is possible to have the API key replaced.

Please email any API key change requests or other security issues to `security@wpgodspeed.io`. This email is delivered to a small security team. Your email will be acknowledged within one business day.

Also, please be sure to use a descriptive subject line for your email.
